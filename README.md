# SDLC & Agile Requirements Engineering Portfolio

Personal portfolio project demonstrating end-to-end requirements engineering for **Titans InstaFood**, a sample B2C food ordering platform. Version 4.0 (2026-05-24).

This repository contains specification artifacts only (PDFs, Excel files, and diagrams). No source code is included.

## What this portfolio demonstrates

- Business and delivery scoping across SDLC initiation and planning artifacts
- Structured functional and non-functional requirements specification
- End-to-end traceability from requirements to stories and system test cases
- Implementation-ready design handoff through architecture, data, and API contracts
